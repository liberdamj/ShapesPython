# ShapesPython
Python Class defining different shapes and properties with them.

Shapes Included
###############
Circle
Triangle
Rectangle


Package Name: shape
Classes:
    - circle
    - triangle
    - rectangle

######
CIRCLE
######

Constructor:
circle.Circle(radius)

Methods:
    - getRadius
    - getDiameter
    - getCircumference
    - getArea

########
TRIANGLE
########

Constructor:
triangle.Triangle(sideOne, sideTwo, sideThree)

Methods:
    - getSideOne
    - getSideTwo
    - getSideThree
    - getPerimeter
    - getArea

#########
RECTANGLE
#########

Constructor:
rectangle.Rectangle(width, height)

Methods:
    - getWidth
    - getHeight
    - getArea
    - getPerimeter
    - isSquare
